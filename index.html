RiseBlog: This is blog web application that enables visitors read various articles posted by authenticated users. Authenticated users are permitted to compose and post articles on the blog. It also allows visitors comment on individual blog posts of their choice and has a built in search functionality making finding articles of interest easy for visitors. It was developed using React.js on the client side and Nodejs on the serverside. Data storage was implemented using MongoDB Visit

UsersGallery: This is a web application that enables visitors browse through a list of random users contained in individual profile cards. It has a built in search functionality that enables vistiors to search for individual profiles by their username. Selection of favorite profiles can also be done by authenticated logged in visitors and profiles can also be un-selected simply by toggling the button that implements the selection feature. The application was built with React.js and user data was implement through API integration. Visit

sparksLP: sparksLP is a seven sectioned responsive landing page built entirely with HTML and SCSS without the use of any CSS or JS framework. It has beautiful user interface features which captures the attention of users upon engagement and interaction Visit

Christ The King Academy: This is a school website built with HTML, SCSS and jQuery. The overall layout was built by utilizing the power of CSS grid which made both responsiveness and architecture of the website easy to carry out. Advanced CSS (SCSS) was used to build each section and some components of the page and the BEM notation was also utilized to make the styling easier and more orderly. Visit
